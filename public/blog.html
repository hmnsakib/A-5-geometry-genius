1. What is DOM and what is the purpose of DOM?

DOM is Document Object Model. It is a JavaScript programming interface that allows interaction or manipulation with HTML content, tag, or attribute.
Its main purpose is to make the website dynamic or more interactive. When a button will be clicked or hovered or some other event will happened then the dynamic website needs some changes or updates. At that moment JavaScript handles those events using DOM to make changes on the HTML file.

2. How to select HTML elements using DOM? Name the DOM methods.

To select an HTML element I have to use the 'document' object of JavaScript. Name the DOM methods are:

  • getElementById()
  • getElementsByClassName()
  • getElementsByName()
  • getElementsByTagName
  • querySelector()
  • querySelectorAll()

3. What is event bubble?

Bubble event is an HTML method that refers to a chain of action in HTML content. When an event occurs on a specific element, will also occur in its parent element up to the root element of the document. This is the main concept of the Bubble event.

4. What is a callback function and why will you use it?

When we pass a function as an argument of another function than the first function is called a callback function.
To make the code more clean, organized, and shorter we should use a callback function. We can do the same thing again but not repeat the same type of code; by using the callback function.
